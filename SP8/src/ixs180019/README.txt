Short Project 8

Files in the folder;

1. Cuckoo.java
2. CuckooDriver.java
3. Timer.java

How to compile and run:

1. Unzip the folder and generate the class files
2. The main function is inside of the CuckooDriver.java
3. In the terminal, inside the unzipped folder, type following code to compile CuckooDriver.java
	- javac ./ixs180019/CuckooDriver.java
4. Type java ixs180019/CuckooDriver for the code execution. Arguements should be passed in the following order: <No of keys> <Load factor> <No of trails>
